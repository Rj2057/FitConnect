FitConnect project prerequisites (not a Python dependency file)

Backend:
- Java 24 (JDK)
- Apache Maven 3.9+
- PostgreSQL 16+ (if running locally without Docker)

Frontend:
- Node.js 20+
- npm 10+

Optional but recommended:
- Docker Desktop 24+
- Docker Compose v2+
- Git 2.40+

Default ports:
- Backend: 8080
- Frontend: 5173
- PostgreSQL: 5432
